heal(player, strength, cost, groups)

Calls the animation player to create healing particles, and increases player’s health.

Parameters

  • player (player.Player) – The player class, so the method can access health and know its position

  • strength (int) – How much health will be recovered

  • cost (int) – How much mana will be deducted from the player

  • groups (list) – Wich groups the magic particles belong

projectile(player, cost, groups, obstacle_sprites, attackable_sprites)

Calls the Projectile class to create a projectile magic.

Parameters

  • player (player.Player) – The Player object, so the method can know it’s position and access it’s mana

  • cost (int) – How much mana will be deducted from the player

  • groups (list) – Wich groups the magic particles belong

  • obstacle_sprites (pygame.sprite.Group) – Sprite group that contains the obstacles in the scenery

  • attackable_sprites (pygame.sprite.Group) – Sprite group that contains the enemy’s sprites

class magic.Projectile(type, direction, caster_rect, frames, groups, speed, obstacle_sprites, attackable_sprites, animation_player, cast_from_center)

Bases: entity.Entity

This class creates a sprite in a certain direction, that moves at constant speed in that direction. This class inherits from the Entity class.

animate()

This method is called once per loop to animate the frames over the images in the self.animation list.

die()

This method is called when the Projectile must end.

timer()

This method verifies how long the projectile lasts, and kills it once it reaches the limit time.

update()

This method sets what will be called everytime the game completes a main loop. Basically, it says what will be “updated” in each frame.
